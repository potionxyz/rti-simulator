# References

## academic papers

1. **Wilson, J. and Patwari, N.** (2010) "Radio Tomographic Imaging with Wireless Networks", *IEEE Transactions on Mobile Computing*, 9(5), pp. 621-632. doi: 10.1109/TMC.2009.174
   - foundational RTI paper, introduces the weight matrix model y = Wx

2. **Kaltiokallio, O., Bocca, M. and Patwari, N.** (2012) "Follow @grandma: Measuring and Lacating Targets Using an RF Sensor Network", *IEEE International Conference on Pervasive Computing and Communications*
   - through-wall RTI, regularisation techniques

3. **Zhao, Y. and Patwari, N.** (2013) "Noise Reduction for Variance-Based Device-Free Localization and Tracking", *IEEE Conference on Sensor, Mesh and Ad Hoc Communications and Networks*
   - CSI-based improvements to RTI resolution

4. **Hamilton, B.R., Ma, X., Zhao, Q. and Xu, J.** (2014) "ACES: Adaptive Cluster-based Estimation for Radio Tomographic Imaging", *IEEE Transactions on Signal Processing*
   - 3D RTI with vertical links, 8-node cube configuration

5. **Bocca, M., Kaltiokallio, O., Patwari, N. and Viani, S.** (2014) "Multiple Target Tracking with RF Sensor Networks", *IEEE Transactions on Mobile Computing*
   - noise handling, Tikhonov regularisation in RTI

6. **Siddon, R.L.** (1985) "Fast calculation of the exact radiological path for a three-dimensional CT array", *Medical Physics*, 12(2), pp. 252-255
   - the ray-voxel intersection algorithm used in the forward model

## books

7. **Strang, G.** (2016) *Introduction to Linear Algebra*, 5th edition, Wellesley-Cambridge Press
   - matrix operations, least squares, pseudoinverse, SVD

8. **Feynman, R.** (1963) *Six Easy Pieces*, Basic Books
   - electromagnetic wave behaviour, interference, attenuation

9. **Gleick, J.** (2011) *The Information: A History, a Theory, a Flood*, Fourth Estate
   - Shannon's information theory, signal-to-noise, channel capacity
